## 工具需求
- MT管理器 或者 NP管理器 或任意反编译工具

## 1. 获取APK安装包

1. 通过 **Google Play** 下载并安装991EX应用
2. 打开MT管理器，提取已安装的应用包
3. 处理安装包格式：
   - 如果提取的文件后缀为 `.xapk`，可通过MT管理器直接转换为 `.apk` 文件
   - 若转换失败，可先将后缀修改为 `.apks`，再转换为 `.apk` 格式
4. 完成上述步骤后，即可获得可用于修改的 `apk` 安装包

## 2. 破解开始

### 2.1 去除Google Play校验

1. 打开反编译工具，搜索以下类和方法：
   ```
   com.pairip.licensecheck.LicenseClient.connectToLicensingService
   ```

2. 找到该方法后，在导航栏长按并选择「清空方法」选项，即可去除Google Play的许可证校验。

### 2.2 修改核心校验机制

#### 2.2.1 找到突破口

APP「设置→关于」界面会显示用户状态（Free/Premium），以此为线索：

- 直接搜索字符串 `Premium`，定位到会员状态判断的核心代码块
- 长按导航将目标方法代码转为Java，看到核心判断逻辑：

```java
if (j.c(s4())) {
    str = "Premium";
} else {
    str = "Free";
}
```

这段代码表明，应用通过调用 `j.c()` 方法来判断用户是否为Premium会员。如果返回 `true`，则显示为Premium；否则显示为Free。

#### 2.2.3 修改核心校验方法

##### 2.2.3.1 分析j.c()方法（关键）

**操作提示**：在Dex++编辑器中，找到以下smali代码行：

```smali
invoke-static {v2}, Lspecializerorientation/Ci/j;->c(Landroid/content/Context;)Z
```

长按代码中的 `->c` 部分，在弹出的菜单栏中选择「跳转」选项，可以直接进入 `j.c()` 方法，然后同样在导航栏长按该方法转Java：

现在分析一下 `j.c()` 方法的Java代码核心逻辑：

```java
public class j {
    public static boolean c(Context context) {
        String f;
        File file = new File(context.getFilesDir(), "license");
        if (file.exists() && (f = f(file)) != null && !f.isEmpty()) {
            return c.b(f).equals(h.b(context));
        }
        return false; // 校验失败返回非会员
    }
}
```

**核心分析**：
- 该方法检查应用目录下是否存在有效的 `license` 文件
- 如果文件存在且验证通过，则返回 `true`（Premium状态）
- 如果文件不存在或验证失败，则返回 `false`（Free状态）

**修改点**：将 `return false;` 改为 `return true;`，使应用始终认为用户处于Premium状态。

找到对应的smali代码并进行修改：


**修改后**：
```smali

```

### 2.3 核心逻辑总结

| 项目 | 说明 |
|------|------|
| **原规则** | 校验`license`文件失败则返回`false`（Free状态） |
| **破解方法** | 强制让校验失败分支返回`true`（Premium状态），直接解锁会员功能 |
| **修改效果** | 无论是否有有效的license文件，应用都会显示用户为Premium会员状态 |


## 3. 软件汉化优化

由于原软件部分翻译不够准确，以下是针对常用功能的汉化优化建议：

### 3.1 图表缩放功能汉化

1. 定位文件：`/res/layout/delegate_address_widgetizer_messenger_module_permission.xml`
2. 如果找不到该文件，可以使用高级搜索：
   - 第一栏留空
   - 第二栏「文本包含」输入搜索关键词：`ZoomIn` 或其它关键字

3. 修改以下缩放功能的字符串映射：

| 原字符串  | 新字符串 |
|----------|----------|
| ZoomIn   | 缩小    |
| ZoomOut  | 放大    |
| ZSquare  | 1:1缩放 |
| ZStandard| 标准缩放 |
| ZoomStat | 统计缩放 |

> **图表说明**：修改后，图表缩放功能的按钮将显示为更直观的英文标识，方便用户快速识别和使用。
