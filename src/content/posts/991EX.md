## 卡西欧计算器991EX修改教程

com.pairip.licensecheck.LicenseClient.connectToLicensingService

### 1. 准备工作
- 一台卡西欧计算器991EX
- 一台安卓手机
- 最新版本的[Android Studio](https://developer.android.com/studio)
- 最新版本的[Java Development Kit (JDK)](https://www.oracle.com/java/technologies/javase-downloads.html)

### 2. 反编译991EX应用
- 打开Android Studio，选择“File” -> “New” -> “Import Project”。
- 导航到991EX应用的APK文件所在目录，选择它并点击“OK”。
- Android Studio将自动反编译应用并创建一个新的项目。

### 3. 修改应用代码
- 在Android Studio中，导航到“app” -> “src” -> “main” -> “java” -> “com” -> “pairip” -> “licensecheck”。
- 找到`LicenseClient.java`文件，双击打开它。
- 在`connectToLicensingService`方法中，添加以下代码：
```java
if (BuildConfig.DEBUG) {
    Log.d("LicenseClient", "connectToLicensingService: DEBUG MODE");
}
